# 🌾 AGRINHO 2026 - Tecnologia no Campo

## 📌 Sobre o projeto
Este site foi desenvolvido para o projeto AGRINHO 2026, com o tema de tecnologia no campo.

O objetivo é mostrar como a tecnologia pode ajudar a agricultura a economizar recursos, aumentar a produção e melhorar a eficiência.

---

## 🧠 Conteúdos abordados
- Matemática aplicada na agricultura
- Programação e automação
- Internet das Coisas (IoT)
- Uso de dados no campo

---

## 🎮 Funcionalidades
- Site interativo
- Botão com mensagem
- Quiz com pontuação
- Interações simples em JavaScript
